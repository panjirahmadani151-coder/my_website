# Website Sederhana 🎉

Website sederhana yang dibuat dengan HTML, CSS, dan JavaScript murni (Vanilla JS).

## 📋 Fitur

- ✅ **Responsive Design** - Website bekerja sempurna di semua ukuran layar
- ✅ **Smooth Scrolling** - Navigasi yang halus ke setiap section
- ✅ **Modern UI** - Desain modern dengan gradient dan animasi
- ✅ **Interaktif** - Form kontak dengan validasi
- ✅ **Animasi** - Smooth fade-in saat halaman di-scroll
- ✅ **No Dependencies** - Vanilla HTML/CSS/JS tanpa library eksternal

## 🏗️ Struktur File

```
my_website/
├── index.html      # File HTML utama
├── style.css       # Styling dan responsive design
├── script.js       # JavaScript untuk interaktivitas
└── README.md       # Dokumentasi (file ini)
```

## 📱 Sections dalam Website

1. **Navigation Bar** - Menu navigasi sticky
2. **Hero Section** - Landing page yang menarik
3. **About Section** - 4 fitur utama dengan card design
4. **Services Section** - 4 layanan bisnis
5. **Contact Section** - Form untuk mengirim pesan
6. **Footer** - Social links dan copyright

## 🎨 Teknologi yang Digunakan

- **HTML5** - Struktur semantic
- **CSS3** - Styling dengan Flexbox, Grid, dan animasi
- **Vanilla JavaScript** - Interaktivitas tanpa framework

## 🚀 Cara Menggunakan

### 1. Buka di Browser Langsung
Klik kanan pada `index.html` → Open with → Browser

### 2. Menggunakan Live Server (VS Code)
- Install extension "Live Server"
- Klik kanan pada `index.html` → "Open with Live Server"

### 3. Clone Repository
```bash
git clone https://github.com/panjirahmadani151-coder/my_website.git
cd my_website
# Buka index.html dengan browser
```

## 🔧 Customization

### Mengubah Warna
Edit file `style.css` di bagian `:root`:

```css
:root {
    --primary: #667eea;
    --primary-dark: #764ba2;
    --accent: #ff6b6b;
}
```

### Mengubah Font
Edit `font-family` di `body` selector dalam `style.css`

### Menambah Navigation Menu
Edit bagian `.nav-menu` di file `index.html`

## 🐛 Troubleshooting

**Website tidak tampil dengan benar?**
- Pastikan semua file (index.html, style.css, script.js) ada di folder yang sama
- Refresh browser dengan Ctrl+Shift+R (hard refresh)
- Buka browser console untuk melihat error

**Form tidak berfungsi?**
- Pastikan script.js di-load dengan benar
- Buka browser console untuk error messages
- Pastikan form elements memiliki class `form-input`
